README how to

WorkAdventure is a hilarious name for a program/system made by "TheCodingMachine". I think they're a French company, but the software is an open-source proximity video/audio chat tool. It uses a browser game engine with open-source tile map format to render the 2D space, then routes video chat through the main Jitsi server.

  • You join the link, enter a name, pick or make an avatar, then join.
  • You can then walk around with arrow keys or WADS. Control video/audio/screen-share with buttons on lower right.
  • IF ANYTHING FAILS or if it just goes black, you can always just reload the same URL and you'll get back fine.
  • If you're near other people, it'll show you video, circle of listening, and a text chat on the left.
  • Different areas trigger different things. Near the fires it launches a Jitsi meet for people in that area. People who are not on the map can join these (I think), they are meet.jit.si meetings at
